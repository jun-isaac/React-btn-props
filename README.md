## 🖼 메인프레임

------------------
## 📕 배운 것

* 내가 만든 함수 안에 인자들을 넣고 그 인자를 활용해 버튼, div 등 안에 텍스트 변경이 가능하다.
* 다른 버튼에서 같은 스타일을 한번에 적용할 수 있다.
* 인자를 활용해 if, else문도 제작이 가능하다.
